# Prison World: Devil's Lock

A vanilla HTML/CSS/JS prototype for a 2D platformer onboarding flow and Level 1 playable scene using lucide icons, ASCII tiles, and emoji assets.

## Run locally

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Deploy on Vercel

1. Create a new GitHub repo and push this project.
2. Go to https://vercel.com/new and import the GitHub repo.
3. Framework preset: **Other**.
4. Build Command: **leave empty**.
5. Output Directory: **leave empty** (Vercel will serve the root).
6. Click **Deploy**.

## Controls

- Move: Arrow keys / A, D
- Jump: W / Space
- Shoot: J
- Pause: P
